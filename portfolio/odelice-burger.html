Challenges

Défis techniques résolus

  • Cohérence visuelle web & mobile

    Maintenir une identité graphique unifiée entre deux codebases distincts (React web et React Native). Résolu grâce à un design system défini sur Figma avec des tokens de couleurs, typographies et espacements partagés.

  • Navigation mobile intuitive avec React Navigation

    Concevoir une navigation naturelle sur l'app (accueil, carte, panier, profil) sans surcharger l'interface. Implémentée avec React Navigation via une bottom tab bar personnalisée adaptée à la charte du restaurant.

  • Performance des images sur le site web

    Les visuels des burgers étant très lourds, le temps de chargement était dégradé sur mobile. Résolu par la conversion en WebP, le lazy loading natif et des skeleton loaders pour améliorer le ressenti utilisateur.

  • Compatibilité iOS & Android avec Expo

    Certains composants React Native se comportaient différemment selon la plateforme. Utilisation des API Platform et des styles conditionnels pour garantir une expérience identique sur iOS et Android sans dupliquer le code.
